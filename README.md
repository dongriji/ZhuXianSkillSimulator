## 简介
根据多玩诛仙12职业加点模拟器修改扩展而来的加点模拟器，目前已经支持全部4个种族19职业加点，并且更新了大量技能描述，修复了大量错误。以后考虑加入星宿技能、三代技能之类的描述。
<br/><br/>
## 支持职业
青云、鬼王、合欢、天音、鬼道、焚香  

九黎、烈山、怀光、天华、太昊、辰皇  

牵机、英招、青罗、破军、归云、画影  

逐霜
